# 🤖 Cosmo Base AI Bot

初心者歓迎の宇宙コミュニティ「Cosmo Base」で動作する、
自動質問回答AIボットです。

宇宙に詳しくない人でも、気軽に質問できる環境をつくることを目的としています。

---

## 🌌 コンセプト

Cosmo Baseは
**「宇宙を身近な選択肢にする」** コミュニティです。

このBotは、

* 宇宙を知らない人でも安心して質問できる
* 専門的すぎない、やさしい回答を返す
* コミュニティ内の会話を促進する

ことを目的に設計されています。

---

## ✨ 主な機能

* フォーラム投稿（質問）に自動でAIが返信
* メンションで再回答
* 回答失敗時は内部で最大3回までリトライ
* 回答成功時に自動でタグ整理
* 投稿時間制限（深夜投稿防止）

---

## 🧠 AIの特徴

* 初心者向けのやさしい回答
* 専門用語をできるだけ使わない
* 3〜6文でコンパクトに回答
* 「宇宙に詳しいお兄さん」的なトーン
* 最後は会話を広げる形で締める

---

## 🛠 使用技術

* Node.js
* discord.js
* Google Gemini API（gemini-1.5-flash）
* GitHub Actions（定期実行）

---

## ⚙️ セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/your-repo/cosmobase-ai-bot.git
cd cosmobase-ai-bot
```

---

### 2. 依存関係インストール

```bash
npm install
```

---

### 3. 環境変数設定（GitHub Secrets）

以下を設定してください：

* DISCORD_TOKEN
* GEMINI_API_KEY
* QUESTION_CHANNEL_ID
* AI_REPLIED_TAG_ID
* HUMAN_REPLIED_TAG_ID

---

### 4. 起動

```bash
node index.js
```

---

## ⏰ GitHub Actions（自動実行）

1日2回などのスケジュールで実行可能

```yaml
cron: '0 3,12 * * *'
```

---

## 🧩 動作仕様

### 回答フロー

1. 質問スレッド作成
2. AIが自動で1回回答
3. メンションで再回答可能
4. 成功時 → AIタグ付与
5. 人間回答 → HUMANタグ付与

---

### リトライ仕様

* 失敗時は最大3回まで再試行
* 内部メモリで管理（ユーザーには見えない）

---

### 投稿時間制限

* 平日：7:00〜22:00
* 土日：12:00〜22:00

---

## 🎯 設計思想

このBotは単なる自動回答ではなく、

* 宇宙を「自分ごと化」するきっかけをつくる
* コミュニティ内の会話を促進する
* 初心者が安心して参加できる空気を作る

ための存在です。
